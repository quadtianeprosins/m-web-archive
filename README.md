# m-web-archive

本仓库用于归档和发布多个独立 HTML 页面。主要用途是保存各类网页的静态快照，便于日后回顾或查阅。

## 目录说明

- `pages/`：存放归档的 HTML 文件，按日期或主题子目录组织。
- `assets/`：页面引用的资源文件（如图片、样式表、脚本等），与对应页面保持相对路径。
- `index.html`：仓库首页，提供归档页面的索引和简要说明。
- `README.md`：本文件，仓库使用说明。

## 页面归档说明

每个归档页面均为独立的 HTML 文件，尽量保留原始内容和基本样式。归档时遵循以下原则：

- 仅保存静态内容，不包含动态交互功能（如登录、实时数据等）。
- 页面资源（图片、CSS、JS）尽量内嵌或随同归档，确保离线可浏览。
- 归档文件命名规则：`YYYY-MM-DD-简短描述.html`，便于按时间排序。

## 维护说明

- 定期添加新的归档页面，更新 `index.html` 索引。
- 如发现归档页面有显示问题或链接失效，可提交 Issue 或 Pull Request。
- 本仓库不承担任何页面内容的法律责任，归档仅供个人学习与参考。

## 使用方式

1. 克隆或下载本仓库。
2. 直接打开 `index.html` 或 `pages/` 下的任意 HTML 文件，即可在浏览器中查看。
3. 也可通过 GitHub Pages 等静态托管服务在线浏览（需自行配置）。
